# 🎯 Desafio Criativo: Engenharia de Prompts para Clientes Bancários

Repositório criado para o Desafio Criativo da DIO (Digital Innovation One), com o objetivo de estruturar um prompt de Inteligência Artificial capaz de extrair insights valiosos a partir de feedbacks de clientes bancários.

---

## 🧱 O Prompt Estruturado

> **Papel:** Atue como um especialista em Experiência do Cliente (CX) focado em operações bancárias.
> **Tarefa:** Analisar comentários de clientes sobre o atendimento via chat do banco para identificar gargalos no atendimento e motivos de insatisfação.

### 📋 Contexto
A análise será usada por gestores de suporte para diminuir o tempo de espera e melhorar as respostas do chatbot. O objetivo é transformar reclamações em melhorias práticas de processo.

### 📊 Dados Disponíveis para a IA
Serão fornecidos blocos de texto contendo:
* ID do cliente;
* Texto do feedback;
* Tempo de espera na fila;
* Status do problema (resolvido/não resolvido).

### ⚙️ Instruções de Análise
1. **Classificação:** Classifique os feedbacks por grau de irritação (Alto, Médio, Baixo) e pelo motivo da queixa (ex: robô travado, demora na transferência para humano, etc).
2. **Padrões:** Identifique os padrões mais repetidos e os 3 problemas mais críticos.
3. **Ações:** Sugira ações práticas e imediatas para a equipe de treinamento de suporte.

### 📦 Formato Esperado da Resposta
Entregue uma lista com os 3 problemas mais críticos encontrados e, para cada um, apresente uma recomendação de solução em formato de tópicos.

### ⚠️ Restrições e Cuidados
* Use apenas os dados fornecidos.
* Não invente dados ou conclusões sem evidências nos textos.
* Proteja dados sensíveis e anônimos.
* Use linguagem clara, executiva e voltada para ação.
